# Clean Todo - React JS

A simple task manager app.<br/>
Check the [Demo](https://clean-to-do.netlify.app/)

## Features

- Select multiple tasks ✅.
- Filter tasks ✅.
- Adding a description with task's title ✅.

## Tools Used

- React JS.
- Typescript.
- Tailwind CSS.
- Chakra UI.
- Redux Toolkit.
- Local Storage.
